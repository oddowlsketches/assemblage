Assemblage Test - Improvements

1. Lazy Loading

Images are now loaded on-demand instead of all at once. This should significantly reduce Supabase egress costs.

  • Only loads metadata initially
  • Loads specific images when needed for each collage
  • Caches up to 50 images in memory
  • Uses placeholder in development mode

2. Improved Paired Forms

The Paired Forms template now uses design principles for better compositions:

  • Golden ratio proportions
  • Visual hierarchy (primary, secondary elements)
  • Modular grid alignment (8px grid)
  • Complementary shape pairings
  • Proper edge relationships
